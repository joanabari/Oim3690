# Week 10 – Log (March 30- April 5)

## What I learned this week
How to build a functional grade calculator using JavaScript functions
How to use if/else conditionals to return letter grades (A–F)
How to convert user input from strings to numbers using Number()
How to validate user input (checking for empty input and ensuring values are between 0–100)
How to use event listeners (click) to trigger actions
How to dynamically update the DOM using textContent
How to change styles dynamically (e.g., color-coding results based on grade)

## Challenges I faced
Making sure user input was properly validated (especially handling empty or invalid values)
Remembering to convert input values to numbers before performing logic
Debugging issues where the event listener didn’t trigger correctly
Getting the DOM updates to display the correct result consistently
Styling results dynamically without breaking the functionality
