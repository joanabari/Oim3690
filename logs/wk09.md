# Week 9 – Log (March 23- March 29)

## What I learned this week
This week I learned the basics of JavaScript, including variables, primitive types, and how to use let and const. I also learned how functions work, how to return values, and how to use operators like comparison and logical operators. Additionally, I learned how JavaScript interacts with HTML through the DOM and how to change content on a webpage using querySelector. This week I also finalized my mini project 2 which is a finance tracker, I am really proud of it.

## Challenges I faced
One challenge I faced was understanding when to use template literals versus regular strings, especially with ${} interpolation. I also found it confusing to remember the difference between == and === and why strict equality is preferred. Another challenge was getting comfortable with how the DOM works and how JavaScript can modify elements on a page.
